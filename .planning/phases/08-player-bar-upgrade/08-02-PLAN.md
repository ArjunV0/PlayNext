**CRITICAL: Preserve E2E selectors.** The tests use:

- `getByRole("button", { name: /auto-play/i })` -- KEEP both auto-play aria-labels

Add `data-testid="play-button"` to the play/pause button.

**Play button size increase:**
Change from `size-9` (36px) to `size-12` (48px) -- this is ~1.33x larger than the current 36px. The other controls stay at `size-8` (32px), making the play button clearly the primary action at 1.5x the skip button size.

Also increase the icon SVGs inside PlayIcon and PauseIcon from `size-4` to `size-5` to match the larger button.

**Ambient gradient background:**
Replace the fixed `bg-gradient-to-r from-blue-500 to-violet-500` with a dynamic gradient that adapts to the ambient color:

```tsx
style={{
  background: `linear-gradient(135deg,
    hsl(var(--ambient-h, 271), var(--ambient-s, 81%), var(--ambient-l, 56%)),
    hsl(calc(var(--ambient-h, 271) + 30), var(--ambient-s, 81%), calc(var(--ambient-l, 56%) - 5%)))`,
}}
```

This creates a gradient from the ambient color to a hue-shifted version (+30 degrees). The fallback values (271, 81%, 56%) match Phase 7's FALLBACK_HSL violet.

Keep the white text/icon color and shadow classes. Update shadow color from `shadow-blue-500/20` to a more neutral `shadow-violet-500/20`.

**Pulse-glow outer ring:**
When `isPlaying` is true, add the `animate-pulse-glow` class (from Phase 6 CSS foundation) and set `--glow-color` to the ambient color:

```tsx
<button
  onClick={togglePlay}
  data-testid="play-button"
  className={`flex size-12 items-center justify-center rounded-full text-white shadow-lg shadow-violet-500/20 transition-all hover:brightness-110 ${
    isPlaying ? "animate-pulse-glow" : ""
  }`}
  style={{
    background: `linear-gradient(135deg, hsl(var(--ambient-h, 271), var(--ambient-s, 81%), var(--ambient-l, 56%)), hsl(calc(var(--ambient-h, 271) + 30), var(--ambient-s, 81%), calc(var(--ambient-l, 56%) - 5%)))`,
    "--glow-color": `hsla(var(--ambient-h, 271), var(--ambient-s, 81%), var(--ambient-l, 56%), 0.4)`,
  } as React.CSSProperties}
  aria-label={isPlaying ? "Pause" : "Play"}
>
  {isPlaying ? <PauseIcon /> : <PlayIcon />}
</button>
```

The `pulseGlow` keyframe (from Phase 6) animates `box-shadow` between `0 0 15px` and `0 0 30px` using `--glow-color`. By setting `--glow-color` to the ambient HSL, the glow adapts per song.

When `isPlaying` is false, the `animate-pulse-glow` class is removed -- instant stop, no fade-out.

**Keep ALL existing functionality:** spacebar toggle, volume slider, next button, auto-play toggle, all event handlers and keyboard listener. Do not modify any button other than play/pause.
